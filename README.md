# Open source section:

How to start bot?
```
node ./src/bot.js
or
npm run start
```
How to Download?
```
git clone https://github.com/Jacob22092/RandomMemes-BOT
```

```

DISCORD_BOT_PREFIX=YOUR-PREFIX
```
